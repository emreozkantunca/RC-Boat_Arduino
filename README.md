# Arduino RC Boat Controller

This project implements a basic Arduino control system for an RC boat.

## Components
- Arduino
- Motor driver
- Servo
- RC receiver

## Features
- Motor control
- Servo steering
- Basic signal processing

## Files
- Boat_Arduino_Codes.ino → main control program
- StopLight.ino → LED test program

## Future Improvements
- Add ESC calibration
- Add telemetry
- Improve motor control algorithm
